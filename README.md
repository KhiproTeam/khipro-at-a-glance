# khipro-at-a-glance
Khipro Keyboard layout at a glance with a single image

# How to Open?
Any image viewer can open the SVG file.

# How to Edit and/or Contribute?
The final SVG file was created as follows:
1. First the layout table was written in LibreOffice Writer.
2. Then the document was exported as a PDF.
3. Then the PDF document was imported in Inkscape using the 'Cairo import' mode for PDFs, provided for PDF import in Inkscape.
This SVG file was created using **Inkscape**. Open the SVG file with **Inkscape**, edit it and then push your contribution to this repository.
